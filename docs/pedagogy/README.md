# Pedagogy

This directory contains instructional, educational, and explanatory
materials related to the Intelligent Cybersecurity Engine (ICE)
Reference Architecture.

Pedagogical materials are intended to support learning, discussion,
and adoption. They do not define or modify the normative architecture.

---

## Relationship to the Reference Architecture

- Normative architecture and governance artifacts live at the
  repository root and under `/architecture-amendments/`.
- Content in `/pedagogy/` explains, teaches, and illustrates those
  artifacts but does not supersede them.

If a conflict exists between pedagogical materials and the normative
text, the normative text SHALL govern.

---

## Intended Audience

Pedagogical materials may be used by:
- engineers
- security practitioners
- students
- executives
- auditors and reviewers
