# Pedagogical Media

This directory contains media assets used for teaching and explaining
the ICE Reference Architecture.

Media may include:
- video walkthroughs
- recorded briefings
- audio narratives
- slide presentations
- transcripts

---

## Authority Notice

Media in this directory is explanatory only.

Normative definitions, requirements, and architectural constraints
are defined exclusively in the reference architecture and its
amendments.

Media SHALL NOT be treated as a system of record.

---

## Versioning Guidance

Media may be versioned to align with:
- architecture releases
- amendment numbers
- instructional cohorts

Where possible, media SHOULD reference the specific amendment or
architecture version it explains.

---

## Accessibility

Where practical, transcripts or text summaries SHOULD accompany
audio or video materials.
